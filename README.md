# Array Exercises

## Instructions

1. Fork & clone to your local computer.
1. Change directory into your newly cloned repo.
1. Install and Start using `npm`

```sh
# Clone
git clone <insert your git clone url here>
cd array-exercises
npm install
npm start

```
